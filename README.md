# Programa para receber dois números e então realizar uma operação matemática como adição, subtração, multiplicação ou divisão de acordo com a escolha do usuário.

Programa que solicita dois números e realiza o cálculo de acordo com a escolha do usuário entre as operações matemáticas disponíveis: adição, subtração, multiplicação e divisão.

Desenvolvido utilizado JDK 1.8 e NetBeans 13.
